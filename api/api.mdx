# API Class

The main API wrapper for MadelineProto. This is the primary entry point for interacting with the Telegram API.

## Constructor

<ParamField path="session" type="string" required>
  Session name for storing session data
</ParamField>

<ParamField path="settings" type="SettingsAbstract">
  Optional settings object for configuring MadelineProto
</ParamField>

```php
use danog\MadelineProto\API;
use danog\MadelineProto\Settings;

$settings = new Settings;
$MadelineProto = new API('session.madeline', $settings);
```

## Methods

### getWebAPITemplate

Obtain the API ID UI template.

<ResponseField name="return" type="string">
  The API ID UI template
</ResponseField>

```php
$template = $MadelineProto->getWebAPITemplate();
```

### setWebApiTemplate

Set the API ID UI template.

<ParamField path="template" type="string" required>
  The template to set
</ParamField>

```php
$MadelineProto->setWebApiTemplate($customTemplate);
```

### startAndLoopMulti

Start multiple instances of MadelineProto and event handlers (enables async).

<ParamField path="instances" type="array<API>" required>
  Array of MadelineProto API instances
</ParamField>

<ParamField path="eventHandler" type="array<class-string>|class-string" required>
  Event handler class name(s)
</ParamField>

```php
use danog\MadelineProto\API;

$instances = [
    'bot1' => new API('bot1.madeline'),
    'bot2' => new API('bot2.madeline'),
];

API::startAndLoopMulti($instances, MyEventHandler::class);
```

## Usage Example

```php
use danog\MadelineProto\API;
use danog\MadelineProto\Settings;

// Create settings
$settings = new Settings;
$settings->getAppInfo()
    ->setApiId(12345)
    ->setApiHash('0123456789abcdef0123456789abcdef');

// Initialize API
$MadelineProto = new API('session.madeline', $settings);

// Start the bot
$MadelineProto->start();

// Make API calls
$me = $MadelineProto->getSelf();
echo "Logged in as: {$me['first_name']}\n";
```
